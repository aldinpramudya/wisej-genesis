# Wisej Fire Range
***
## Prerequisites
***
- Visual Studio with ASP.NET and Web Developmen & .NET Dekstop Development Workload
- Wisej.NET Extensions
- .NET 8.0 / .NET 10.0 SDK

## Getting Started
***
1. Clone the repository
2. Rebuild Solution with delete obj, bin, .vs folders (only if error when debugging) 
3. Run
> dotnet restore

***
